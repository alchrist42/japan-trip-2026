# Визы и перелеты

## Нужно разобрать

- Виза в Японию.
- Виза во Вьетнам / въезд через Нячанг, если лететь из Японии.
- Перелет Kansai/Itami/Kobe -> Okinawa.
- Перелет Okinawa -> Вьетнам или Okinawa -> Tokyo/Osaka -> Вьетнам.
- Возврат второй пары в Москву.

## Известно

- В Японию прилет: Tokyo Narita, 19 SEP 2026, 20:55, из Shanghai.
- После Японии одна пара летит во Вьетнам, вторая - в Москву.
- Багаж: предпочтение ручной клади, хотя багаж может быть включен в билеты.

## Проверить

- Ограничения ручной клади на внутренних японских перелетах.
- Нужен ли сдаваемый багаж на сегменте во Вьетнам.
- Какой аэропорт Kansai-региона удобнее для перелета на Okinawa после Kyoto/Osaka.
