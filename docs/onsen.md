# Onsen / сауны

## Пожелания

- Интересны традиционные японские бани/onsen.
- Совместные или private варианты подходят.
- Татуировок нет.

## Проверить

- Private onsen / family bath для пары или компании.
- Нужны ли купальники.
- Правила бронирования.
- Можно ли совместное посещение для mixed-gender группы.
- Удобство по маршруту: Tokyo, Kyoto или Okinawa.
