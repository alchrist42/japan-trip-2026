# Japan Trip Context

Короткий рабочий документ по поездке. Не раздувать без необходимости: фиксировать только факты, решения, важные предпочтения и чеклист вопросов.

## Факты

- Едут две пары, всем около 30 лет.
- Прибытие: Tokyo Narita, 19 SEP 2026, 20:55, рейс из Shanghai.
- Примерный срок Японии: 19 SEP 2026 - около 03 OCT 2026.
- Основной маршрут: Tokyo 19-24 SEP 2026, Kyoto/Kansai 24-28 SEP 2026, Okinawa 28 SEP - 03 OCT 2026.
- Маршрут утвержден.
- После Японии:
  - пользователь с девушкой летят во Вьетнам: Нячанг/Далат/возможно Дананг, детали позже;
  - вторая пара возвращается в Москву и билеты покупает сама.

## Предпочтения

- Формат: без жесткого туристического расписания, основные планы можно добирать на месте.
- Tokyo: хочется что-то связанное с Pokemon Go / Pokemon.
- Бани/сауны: интересны традиционные японские варианты, совместные/private варианты подходят, обсудить отдельно.
- Okinawa: главный остров, пляжный эпизод; более важен для второй пары, потому что первая пара потом еще едет во Вьетнам.
- Жилье: средний бюджет, но можно дешевле или дороже, если это оправдано.
- Жилье для 4 человек: общий домик/номер/апартаменты на 4-6 мест даже предпочтительнее.
- Ориентир по жилью: оптимально до $100 за ночь за четверых.
- Жилье можно искать через Trip.com/Библио-Глобус/российские сервисы; Booking/Tripadvisor технически возможны через вьетнамскую Visa, если вариант заметно лучше.

## Логистика

- Татуировок нет.
- Водительские права: вероятно только у пользователя; международные категории A, B, но нужно проверить наличие/актуальность.
- Багаж: скорее всего без сдаваемого багажа, несмотря на включенный багаж; предпочтение ручной клади.
- Техника: возможно взять девушке легкий ноутбук вместо игрового рабочего.

## Основной маршрут

- 19 SEP 2026: прилет в Narita в 20:55, заселение в Tokyo.
- 20-24 SEP 2026: Tokyo и окрестности.
- 24-28 SEP 2026: Kyoto/Kansai, база Kyoto; Osaka как дневной/вечерний выезд.
- 28 SEP - 03 OCT 2026: Okinawa, главный остров; районы-кандидаты Naha/Ginowan/Chatan.
- Около 03 OCT 2026:
  - пользователь с девушкой: вылет во Вьетнам;
  - вторая пара: вылет в Москву.

## Закрытые решения

- Kansai-база: Kyoto; fallback - Osaka/Namba, если жилье в Kyoto будет сильно хуже.
- Okinawa: главный остров; не Ishigaki/Miyako для этой поездки.
- Машина на Okinawa: не обязательна на весь срок; можно рассмотреть аренду на 1 день после проверки прав и цены.
- Onsen-ryokan: не обязательная часть маршрута; Arima Onsen оставить optional на 27 SEP.
- JR Pass: по умолчанию не брать; пересчитать только если добавятся длинные железнодорожные выезды.
- Вылет дальше планировать с Okinawa как стартовой точки, со стыковками по цене и расписанию.
