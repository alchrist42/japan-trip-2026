# Codex Project Rules

- This project is for planning the Japan/Vietnam trip.
- Before answering trip-planning questions, read `TRIP_CONTEXT.md` and use it as the source of project context.
- Keep `TRIP_CONTEXT.md` compact: facts, decisions, preferences, checklist items, and important open questions only.
- When a conversation produces a durable decision or important new fact, update `TRIP_CONTEXT.md` briefly.
- Public/shareable trip materials live in `docs/`; keep them suitable for the temporary GitHub Pages site and useful to friends.
- Keep `docs/` focused on route, decisions, shortlists, open questions, and checklists. Do not add raw research logs, temporary reasoning, booking numbers, passport/visa documents, tickets, or exact accommodation addresses.
- Keep working research, agent context, and update instructions in root-level Markdown files unless the user asks to publish a summarized version.
- Do not add long research notes, full explanations, or temporary reasoning to `TRIP_CONTEXT.md`; summarize only the outcome.
- Answer the user in Russian unless they ask otherwise.
